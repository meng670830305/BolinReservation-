介绍：当你修改一个 HTML/XML 标签时，这个插件会自动帮你修改对应的闭合标签。
使用方式：安装后，当你更改任何标签的名称时，它的配对标签也会自动更新。

ESLint 插件主要用来检测代码的语法格式，以便我们规范书写，避免因编译所发生的错误

Prettier - Code formatter 插件
